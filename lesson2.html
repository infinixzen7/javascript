Lesson 2 : Variables

Beginner

📘 What are Variables?

Variables are containers used to store information inside a JavaScript program. Instead of writing the same value repeatedly, we store it inside a variable and use it whenever required.

let name = "Rahul"; console.log(name);

💡 Why Do We Use Variables?

Variables make programs easier to write and manage. They allow us to store numbers, text, true/false values and many other kinds of data.

let age = 20; let city = "Delhi"; let isStudent = true;

📌 Types of Variables

JavaScript provides three ways to declare variables.

var language = "JavaScript"; let score = 95; const pi = 3.14159;

var → Old method (avoid in modern JavaScript)

let → Value can change later.

const → Value cannot be changed after declaration.

📝 Variable Naming Rules

Follow these important rules while creating variables:

  • ✔ Variable names can contain letters, numbers, _ and $.
  • ✔ Variable names cannot start with a number.
  • ✔ Variable names are case-sensitive.
  • ✔ Do not use JavaScript keywords like let, const, if, for, etc.
  • ✔ Use meaningful names like studentName instead of x.
let studentName = "Amit"; let marks = 95; const country = "India";

🚫 Invalid Variable Names

The following declarations are incorrect.

let 1name = "Rahul"; ❌ let my-name = "Rahul"; ❌ let const = 20; ❌

These produce errors because they break JavaScript naming rules.

✅ Correct Variable Names

let firstName = "Rahul"; let totalMarks = 450; let user_age = 21; const PI = 3.14159;
