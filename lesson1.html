Lesson 1: Introduction to JavaScript

Beginner

📘 What is JavaScript?

JavaScript is a programming language used to make websites interactive. It runs in the browser and helps create dynamic features like buttons, forms, animations, games, and more.

🌐 Where is JavaScript used?

- Websites (Frontend)
- Servers (Node.js)
- Mobile apps
- Games

💡 First JavaScript Example

console.log("Hello World");

This prints "Hello World" in the browser console.
